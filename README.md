Hirex   

- Hirex is a job searching web application that helps users find relevant job opportunities based on their preferred job title and applied filters.  
- The platform fetches job listings from multiple job sources and displays them in one place.  
- When a user clicks on the Apply option, the application redirects the user to the official job platform to complete the application process.  


Features:  

- Search jobs using specific job titles  
- Apply filters to refine job results  
- Fetch job listings from multiple job platforms  
- View detailed job descriptions  
- One-click redirection to apply on official job sites  
- User-friendly interface  
- Fast and responsive performance  



Technologies Used  

- HTML  
- CSS  
- JavaScript

open index.html



How the Application Works  

- The user enters a job title in the search bar.  
- The user applies filters such as location, job type, etc.  
- The application fetches job details from multiple job platforms.  
- The job results are displayed to the user.  
- The user clicks on Apply for a selected job.  
- The system redirects the user to the official job website.  
- The user completes the job application on that platform.
